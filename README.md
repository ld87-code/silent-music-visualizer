## Features

- Three vibes: **Lo-fi (1), Ambient (2), Techno (3)**  
- Click/tap anywhere for **particle bursts**  
- Optional **auto-cycle demo mode** (press 4 to toggle)  
- Responsive canvas: works on desktop and mobile  
- Cinematic trails, pulsing circles, and rhythmic ribbons  

## How to Use

1. Open the live demo or clone the repository.  
2. Press **1 / 2 / 3** to change vibes manually.  
3. Click/tap anywhere to trigger particle bursts.  
4. Press **4** to toggle **auto-cycle mode**.
